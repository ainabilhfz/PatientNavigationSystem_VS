Patient Navigation System is a system consist of Model, View and Controller. 
The presentation tier is the front end layer for Patient Navigation System 
which accessible through a web browser or web-based application. 
It displays content and information for the end user namely Community, Nurse and Doctor. 
Controller contains the functional business logic which includes registration and the system management. 
Model compromises the data access layer for the system.
